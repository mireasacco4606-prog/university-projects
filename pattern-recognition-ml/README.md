# Pattern Recognition and Machine Learning Project
**Course:** CCE2502 - Semester 2, 2025/2026

## Description
Python-based assignment experimenting with k-Nearest Neighbour (k-NN) and radius-based r-NN classifiers. Includes tasks on model accuracy, probability prediction, and algorithm complexity analysis.

## Tools & Technologies
- Python
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## Key Features
- Implemented k-NN and r-NN classifiers
- Trained models on classification and regression datasets
- Plotted accuracy vs k/r for model selection
- Analyzed time and space complexity using different search algorithms
