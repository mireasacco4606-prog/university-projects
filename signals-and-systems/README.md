# Signals and Systems Project
**Course:** CCE2203 - Semester 1, 2025/2026

## Description
This project focuses on processing audio signals using Python. Techniques implemented include Fast Fourier Transform (FFT) and convolution filtering.

## Tools & Technologies
- Python
- NumPy
- Matplotlib

## Key Features
- Load and analyze audio signals
- Apply FFT to examine frequency components
- Filter signals using convolution
- Plot and visualize results
